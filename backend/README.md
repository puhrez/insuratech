# Backend

This is InsuraTech's backend.


## Local Use

Tools:
* Python3.6


### Make commands
```
$ make
dev-setup                      Install dev reqs
dev                            Run dev server
setup                          Install reqs
shell                          Run flash shell
test-setup                     Install testing reqs
test                           Run tests
```

To use the shell:

```
$ make setup
$ make shell
```

To run the server:

```
$ make setup
$ make dev
```

To run the tests:

```
$ make test
```
